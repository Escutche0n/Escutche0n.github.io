> Despite what television and movies tells us, 

### **Git  Instructions**

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people.



**The basic mechanism is**

1. Initialise the local folder
2. Clone / Pull files from repository to the folder
3. Edit
4. Commit comments for this edit
5. Push the files back to the repository on the server.
